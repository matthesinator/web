JSPs, Servlets, JaxB


JSPs und Servlets basieren auf Java und haben zum Ziel, Websites dynamisch zu erstellen. JSPs, bzw Java Server Pages, sind HTML Dateien, die Java-Code enthalten. Servlets im Gegenzug sind Java-Klassen, die HTML-Code ausgeben.
JaxB wird benutzt, um Java-Objekte nach XML zu übersetzen und anders herum.
